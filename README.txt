The Elmore Calculator

Versions

V.1
- Flat UI
- Supports addition, subtraction, multiplication, and division of whole numbers
- Non fragile


Known Bugs
- Fractional and decimal numbers are not supported
- Dividing integers with a remainder displays a value larger than the display size
- Trying to input '0' or a number with a leading 0 leads to a *handeled* runtime exeption
